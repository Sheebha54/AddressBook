# 📒 Address Book Management System

A menu-driven Address Book application implemented in C using file handling and structures.

---

## 📌 Overview

This project is a simple Address Book Management System that allows users to store and manage contact details.

The application supports adding, searching, editing, and deleting contacts using file storage.

---

## 🚀 Features

- Add new contact  
- Search contact by name  
- Edit existing contact  
- Delete contact  
- Display all contacts  
- Save contacts to file  

---

## 🧠 Core Concepts Implemented

### Structures
- Store contact details (name, phone, email)
- Organized data handling

### File Handling
- Store contacts in file
- Read and update records

### Searching & Sorting
- Linear search implementation
- Efficient data retrieval

---

## 🛠️ Tech Stack

- Language: C  
- Concepts: Structures, File Handling  
- Compiler: GCC
