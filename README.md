# Address Book

## Description
A simple and efficient C-based Address Book application. It uses structures, pointers, file handling, strings, and bitwise operations to manage contacts. Supports creating, editing, deleting, and searching contacts with ease.

## Tech Stack
- C Language
- Structures & Pointers
- File Handling
- Strings
- Bitwise Operations

## Features
- Add new contacts
- Edit existing contacts
- Delete contacts
- Search contacts by name

## How to Run
1. Clone the repository
2. Open in any C compiler (e.g., GCC)
3. Compile and run `main.c`
